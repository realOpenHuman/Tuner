# 音频调谐器 🎵

一个通过麦克风输入实时检测音频频率的乐器调音工具

## 功能特性
- 🎤 **实时分析**: 通过麦克风捕捉环境声音并分析频率成分
- 🎯 **多频检测**: 显示7个主要频率分量及其对应音名（如A4: 440Hz）
- 📊 **强度可视化**: 以百分比条显示各频率相对强度
- 🔇 **噪声门限**: 自动过滤低强度环境噪声（阈值：60%强度）
- 🖥️ **清爽界面**: 自动清屏实现流畅的可视化更新

## 环境要求
- Python 3.6+
- `pyaudio` (需PortAudio支持)
- `numpy`

## 安装指南

# 安装依赖库

# Linux用户需先安装PortAudio：
